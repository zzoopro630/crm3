# 프로젝트명 : insu-analysis
## 목표
- 비교 분석 테이블을 깔끔한 디자인의 A4 사이즈 인쇄물로 출력
- 구조 및 내용 참고 : sample.pdf
## 고려할 사항
- 월 평균 리포트 생성 건수 : 만나는 고객수에 따라, 담당자에 따라 모두 다름                                                                              
- 현재 스프레드시트 인쇄 시 겪는 구체적인 불편함 : A4 사이즈에 가독성 좋게 배치된 비교표를 만드는 어려움
- 보장 항목 구조가 고정인지, 고객마다 다른지 : 항목은 대체로 동일 (업데이트는 비정기적으로 필요)
- 사용자 수 (본인만 사용 vs 팀 전체 사용) : 팀 전체 공유
## 기술스택
- 추천
## 구조
- 프런트엔드 : 사용자가 숫자를 입력하고, PDF 파일 생성 및 다운로드 제공
	- 생성된 PDF 파일은 A4에 딱 맞는 레이아웃으로 세부조정 없이 바로 프린터 출력할 수 있어야 함.
- 백엔드 : 관리자용 화면. 테이블 메인 항목과 세부항목을 수정할 수 있어야 함.
	- 고객명으로 불러오기 기능을 어떻게 구현해야 할까 고민. (데이터베이스를 만들면 규모가 커질 것 같아서)
