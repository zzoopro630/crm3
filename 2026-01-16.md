- 새로운 브랜치 생성
- 수정사항에 대한 이슈 생성하고 계획 수립
#### 버그 수정
- F5 등급, DB관리 메뉴 안보이는 문제.
- 화면 : 라이트모드 기본값으로 적용.
- 내 계정 : 내 계정정보 설정 메뉴 활성화 (이름, 연락처 등), 등록된 이메일은 F1 (최고관리자) 등급만 수정 가능.
####  DB 관리 
- 청약완료 탭 : 페이지네이션 오류. 아무 의미 없는 페이지 나누기 중. 오류 수정.
- DB관리에서 처리되는 고객과 '고객관리'에서 처리되는 고객 데이터는 철저하게 분리되어 있어야 함
	- 문제 : 현재 DB관리에 등록된 고객 정보가 '고객관리' 메뉴에 노출되고 있음
- 본인이 배정받은 고객 이외의 정보는 볼 수 없어야 함. (중요, 다시 확인)
#### 팀 관리
- 이름을 누르면 개인정보(가입정보) 확인할 수 있도록 개선.
- 고객보기 필요없음. (기능 삭제)
- 총 고객, 신규, 연락완료 등 상담현황은 다른 메뉴로 분리 (메뉴명 : 고객현황)
	- 본인 기준 하위에 있는 내용만 조회할 수 있도록 (자기 팀 내용만 조회, F5 등급이라면 자기 상태만 조회)
#### 설정 / 사원관리
- F1 등급 : 사원 정보 수정에서 개인 이메일 주소 수정할 수 있도록 수정 (등록된 사원들 정확한 이메일 변경 필요)
- F2~F5 등급 : 본인 소속(조직)만 보이게 수정. 정보 수정 기능 없음. (F1에 요청해서 수정)
- 사원 관리 메뉴처럼 2단계로 진입한 경우, 메인 메뉴는 아이콘만 남기고 왼쪽으로 축소되도록 디자인 수정.
	- 축소되더라도 어떤 메뉴에 들어와 있는건지 아이콘에 컬러로 표시
- 사원 목록 : 이름, 직급, 상위자, 보안등급, 이메일 순으로 열 순서 변경.
	- 데스크탑 기준, 한 화면에 10명씩 보이도록 페이지네이션 적용.
	- 일괄 수정 기능 적용. (조직, 직급, 상위자 일괄 선택 지정 가능하도록)
- 비활성 사원 목록 : 완전 삭제 기능 추가. (더미 데이터 관리 목적, 테스트 용도 필요)
